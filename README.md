# Setup
> Composer Installation
> [![Download]](https://getcomposer.org/download/) </br>
```bash
composer install
```
> Laravel Installation
```bash
php artisan key:generate
```
```bash
php artisan migrate:fresh --seed
```
```bash
php artisan config:publish cors
```
> TailwindCSS Installation
```bash
pnpm install
```
`Or`
```bash
npm install
```
> Setup .env
```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
```
```bash
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=
DB_USERNAME=postgres
DB_PASSWORD=
```
> Setup php.ini
```bash
extension=pdo_pgsql
extension=pgsql
```

# Running Development
> Laravel
```bash
php artisan serve
```
> TailwindCSS
```bash
pnpm dev
```
`Or`
```bash
npm run dev
```
